# PinkCycle Frontend (v2)

React + Vite single-page app with:

- Email/password accounts
- Pink, butterfly-animated UI
- Dashboard with predictions & calendar
- Period logging & history
- Rich Settings screen:
  - Account & Profile
  - Cycle tracking & prediction options
  - Notifications & reminders
  - Privacy & security
  - Integrations & appearance
  - Data management (export & delete)

## Setup

```bash
cd frontend
npm install
npm run dev
```

Default dev URL: `http://127.0.0.1:5173`
Backend must run at `http://127.0.0.1:8000` (see ../backend).
